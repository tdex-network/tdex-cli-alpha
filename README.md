Wallet CLI for making swaps on TDEX

## Usage

In-depth documentation for using the `tdex-cli` is available at [docs.tdex.network](https://docs.tdex.network/tdex-cli.html)

## Development

### Requirements

* Node/npm or yarn

### Instructions

* Install deps

```sh
yarn install
```

* Build .ts files

```sh
yarn build
```

* Build and Test

```sh
yarn test
``` 

* Try it out locally

```sh
npm install -g 
tdex-cli --version
```
